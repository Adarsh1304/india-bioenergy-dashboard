# India Bioenergy Production & Biomass Analytics Dashboard

## 📊 Project Overview
This project presents an interactive MIS-style analytics dashboard analyzing India's bioenergy production trends and biomass availability using real government datasets.

The dashboard combines global benchmarking with state-level operational analysis to support renewable energy planning and biomass resource monitoring.

---

## 🎯 Business Objective
To analyze:
- Growth trends in India's bioenergy production
- Global production benchmarking
- State-wise biomass availability
- Crop-level biomass contribution

---

## 🧰 Tools & Technologies
- Google Sheets (Data Cleaning & Transformation)
- Looker Studio (Dashboard Development)
- Data Modeling & KPI Design

---

## 📂 Data Sources
- FAOSTAT Bioenergy Database
- Government of India Biomass Atlas

---

## 🔄 Data Preparation Steps
- Cleaned raw government datasets
- Converted wide-format agricultural data into analytical long format
- Created benchmarking index comparing global production with India
- Built KPI metrics and geo-analysis dataset

---

## 📈 Dashboard Features
✅ Executive KPI monitoring  
✅ Bioenergy production trend analysis (2013–2023)  
✅ Global production comparison  
✅ India state-level biomass map  
✅ Interactive crop filtering  

---

## 🔍 Key Insights
- India shows steady bioenergy growth (~2.3% over the decade).
- Northern and central Indian regions demonstrate high biomass concentration.
- Regional crop specialization highlights potential zones for decentralized bioenergy infrastructure.
